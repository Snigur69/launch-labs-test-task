# Launch Labs – Test Task

## About

This project is a test task for Launch Labs.

The application displays a list of athletes and supports:

- Search
- Filtering
- Sorting
- Pagination

---

## Tech Stack

- React
- TypeScript
- Vite
- TailwindCSS

---

## How to Run

### Install dependencies

```bash
npm install
```

## Start development server
```
npm run dev
```
